

# 마인드맵 기반 게시판

## 개요
카테고리와 게시글을 마인드맵으로 시각화하는 자유 게시판. 검색 시 매칭 노드의 부모-자식 경로 + 같은 레벨 형제 노드를 함께 하이라이트하여 맥락을 보여줌.

## 주요 화면

### 1. 마인드맵 뷰 (메인)
- 중앙에 루트 노드, 가지 형태로 카테고리 → 하위 카테고리 → 게시글이 펼쳐지는 인터랙티브 마인드맵
- 노드 클릭 시 내용 미리보기 패널 표시
- 노드 드래그로 위치 조정, 줌/패닝 지원
- React Flow 라이브러리 활용

### 2. 검색 기능
- 상단 검색바에서 제목/내용 검색
- 검색 결과: 매칭 노드를 강조 표시(색상 변경)
- **부모-자식 경로**: 루트부터 매칭 노드까지 경로를 연결선과 함께 강조
- **형제 노드**: 같은 부모 아래의 다른 노드도 반투명하게 표시
- 비관련 노드는 흐리게 처리(fade out)

### 3. 노드 CRUD
- 노드 우클릭 또는 '+' 버튼으로 하위 노드(카테고리/게시글) 추가
- 게시글 노드: 제목, 내용(리치텍스트), 작성일
- 편집/삭제는 노드 선택 후 사이드 패널에서 수행
- 카테고리 노드: 이름, 색상 지정 가능

### 4. 사이드 패널
- 노드 클릭 시 우측에 상세 내용 표시
- 게시글 작성/수정 폼
- 해당 노드의 하위 구조 목록 보기

## 백엔드 (Supabase)
- `nodes` 테이블: id, parent_id, title, content, type(category/post), color, position, created_at
- 트리 구조를 parent_id로 관리
- 검색은 title/content 대상 텍스트 검색
- RLS 정책 적용

## 기술 스택
- React Flow (마인드맵 렌더링/인터랙션)
- Supabase (데이터 저장/검색)
- Tailwind + shadcn/ui (UI 컴포넌트)
